What renewal looks like

A lot like the first time you bought and activated an SSL, probably:

  1. Submit your data to CA
  2. Confirm that you control the domain you wish to secure.
  3. Pass company checks: organization validation for an OV SSL or extended validation for an EV SSL

Tools you may find useful

  1. Check if SSL and Private Key are correct pair — comes in handy in case you’ve created multiple CSRs and Private Keys before to avoid errors during installation.
  2. Confirm SSL is installed properly
  3. Make sure there no CSR errors
  4. Сonvert your SSL files to PFX (for Windows Servers and some other applications)
